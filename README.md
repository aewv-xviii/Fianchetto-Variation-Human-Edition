### Features

* Standalone port of the Fianchetto Variation arsenal
* Removed dependency on Milira Race, Ancot Library, and the original Fianchetto addon
* Requires only Harmony
* Includes CE patch support

### Included Content

* Milian Conductor Rod
* Strike drone: Malkuth
* Strike drone: Gevurah
* Strike drone: Keter
* Pulse protection
* Hypercharge

### How to Use

* Equip **Strike drone: Malkuth**, **Strike drone: Gevurah**, or **Strike drone: Keter** as apparel to deploy and use the drone system.
* The drone items can function without the **Milian Conductor Rod** equipped.
* The **Milian Conductor Rod** remains the intended paired weapon and provides the pawn's direct ranged attack plus drone-related bonuses.
* **Strike drone: Malkuth** provides Pulse protection.
* **Strike drone: Gevurah** provides Hypercharge.
* **Strike drone: Keter** is the heavy composite drone set and includes the restored drone work functions.

### Research Placement

* The custom research line was moved to the far-right spacer end of the tree.
* Float unit architecture appears first on the right-side spacer branch.
* Funnel field control is placed immediately to the right of it.
* Composite drone weaponry is placed slightly further up and to the right as the final step.

### Notes

* This port focuses on removing the original Milira-specific dependency chain so the content can run in a general setup with Harmony.
* The original conductor gear set was kept for completeness.
* Milian conductor headset is the practical part of that set in this port.
* Milian conductor cloak is still included as leftover source content, and should be treated as legacy Milian-specific content.
* This was originally made for personal use, so there may still be rough edges or bugs.
* If you run into a specific use case or weird breakage, feel free to leave a comment.
